## How to Run

```bash
git clone https://github.com/dynamicfire/Rocketman-Odyssey.git
```

Open the project in IntelliJ IDEA (or any Java IDE you prefer).

> **Note:** Set JVM max heap size to at least 256MB (`-Xmx256m`)

## Controls

- **F1** — Start game
- **Space** — Pause
